# Boss直聘智能助手

> 🤖 AI驱动的半自动化招聘筛选与接待工具

## ✨ 功能特点

### 🛡️ 安全模式（推荐）
- ✅ **智能筛选简历** - 自动分析候选人，显示匹配度评分
- ✅ **智能回复助手** - AI生成3种风格回复，一键填充
- ✅ **智能打招呼** - 推荐高匹配候选人，生成招呼语
- ✅ **完全合规** - 所有操作需人工确认，不会被检测

### ⚡ 自动模式（可选）
- ⚠️ **全自动回复** - 自动回复应聘者消息（带反检测）
- ⚠️ **全自动打招呼** - 自动向高匹配候选人打招呼
- ⚠️ **存在风险** - 可能被平台检测，请谨慎使用

## 📦 安装步骤

### 1. 下载插件
将整个 `boss-zhipin-assistant` 文件夹保存到本地

### 2. 在Chrome中加载
1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 打开右上角的 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `boss-zhipin-assistant` 文件夹

### 3. 验证安装
看到插件图标出现在浏览器工具栏即表示安装成功 🎉

## 🚀 快速开始

### 第一步：配置插件
1. 点击浏览器工具栏的 🤖 图标
2. 选择运行模式：
   - **安全模式**（推荐）：AI辅助 + 人工确认
   - **自动模式**（慎用）：完全自动化
3. 可选：开启AI智能分析
   - 打开"使用AI智能分析"开关
   - 输入Claude API Key
   - 点击"测试连接"验证
4. 点击"保存设置"

### 第二步：开始使用
1. 登录 [Boss直聘](https://www.zhipin.com/)
2. 进入"沟通"页面查看候选人列表
3. 插件会自动：
   - 在候选人卡片上显示匹配度徽章
   - 为高匹配候选人显示"🤖 AI建议打招呼"按钮
4. 点击候选人进入聊天窗口
5. 当收到新消息时，插件会自动弹出回复建议
6. 选择一个回复，点击"使用此回复"
7. 检查内容后，点击Boss直聘的"发送"按钮

## 📖 功能说明

### 功能1：智能筛选简历
**作用**：自动分析候选人匹配度

**使用方法**：
- 浏览候选人列表时，每个候选人卡片会自动显示：
  - 匹配度分数（0-100分）
  - 星级评价（⭐⭐⭐⭐⭐）
  - 绿色=高匹配 | 橙色=中等 | 红色=低匹配

**评分依据**：
- 学历是否符合要求
- 工作年限
- 技能标签匹配
- 期望薪资
- 在线活跃度

### 功能2：智能回复助手
**作用**：AI生成回复建议，一键填充

**触发条件**：
- 应聘者发来新消息
- 输入框为空

**使用步骤**：
1. 收到消息后，页面自动弹出"AI智能回复建议"面板
2. 查看3种风格的回复：
   - 📋 正式 - 专业严谨
   - 😊 友好 - 轻松亲和
   - ⚡ 简洁 - 直接简短
3. 点击"使用此回复"按钮
4. 内容自动填充到输入框
5. 检查无误后点击发送

**智能识别**：
- 如果问薪资 → 建议"可面聊详谈"
- 如果问岗位 → 简要介绍职责
- 普通咨询 → 友好回复

### 功能3：智能打招呼
**作用**：推荐高匹配候选人，生成招呼语

**触发条件**：
- 候选人匹配度 ≥ 75分
- 自动显示"🤖 AI建议打招呼"按钮

**使用步骤**：
1. 看到高匹配候选人卡片上的绿色徽章
2. 点击"🤖 AI建议打招呼"按钮
3. 弹出面板显示：
   - 候选人匹配度分析
   - 2种风格的招呼语
4. 选择一种，点击"使用"
5. 自动跳转到聊天窗口并填充内容
6. 检查后点击发送

## ⚙️ 高级设置

### AI分析配置
**默认**：使用规则评分（不需要API）
**高级**：开启AI智能分析（需要API密钥）

**获取Claude API Key**：
1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 注册/登录账号
3. 创建API Key
4. 复制 `sk-ant-...` 格式的密钥
5. 粘贴到插件设置中

**AI分析优势**：
- 更准确的匹配度评估
- 更个性化的回复生成
- 更吸引人的招呼语

### 筛选标准自定义
编辑 `utils/config.js` 中的 `filterCriteria`：

```javascript
filterCriteria: {
  minEducation: '大专',          // 最低学历
  minExperience: 0,              // 最低工作年限
  requiredSkills: ['视频剪辑', 'PR'],  // 必备技能
  excludeKeywords: ['兼职', '实习生'],  // 排除关键词
  maxSalaryExpectation: 20000    // 最高期望薪资
}
```

## 🔧 故障排查

### 问题1：插件不工作
**解决方案**：
1. 确认已在Boss直聘页面（https://www.zhipin.com/）
2. 刷新页面（Ctrl/Cmd + R）
3. 查看浏览器控制台（F12）是否有错误
4. 检查插件是否已启用（chrome://extensions/）

### 问题2：看不到匹配度徽章
**原因**：页面DOM结构可能更新
**解决方案**：
1. 等待10秒（插件需要扫描页面）
2. 点击插件图标，点击"刷新分析"
3. 如果仍无效，请查看控制台日志

### 问题3：回复建议不弹出
**原因**：输入框选择器可能变化
**解决方案**：
1. 确保输入框为空（有内容时不会触发）
2. 等待对方发送新消息
3. 手动刷新页面

### 问题4：API连接失败
**检查清单**：
- ✅ API密钥格式正确（sk-ant-...）
- ✅ 网络连接正常
- ✅ API账户有余额
- ✅ 已开启"使用AI智能分析"开关
