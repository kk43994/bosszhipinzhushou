# ⚡ 快速测试指南

## 📋 测试前检查清单

- [ ] Chrome浏览器已安装（版本 >= 88）
- [ ] 有Boss直聘账号（招聘者身份）
- [ ] 文件夹完整（所有文件已创建）

## 🚀 5分钟快速测试

### 步骤1：加载插件（2分钟）

1. 打开Chrome浏览器
2. 地址栏输入：`chrome://extensions/`
3. 打开右上角 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择 `boss-zhipin-assistant` 文件夹
6. 看到插件出现 ✅

### 步骤2：配置插件（1分钟）

1. 点击浏览器工具栏的 🤖 图标
2. 默认配置即可，无需修改
3. 确认显示：
   - ✅ 安全模式（已选中）
   - ✅ 智能回复助手（已开启）
   - ✅ 打招呼助手（已开启）

### 步骤3：测试功能（2分钟）

#### 测试A：智能筛选
1. 登录 [Boss直聘](https://www.zhipin.com/web/chat/index)
2. 进入"沟通"页面
3. 等待5-10秒
4. 预期结果：
   - ✅ 候选人卡片右上角出现 **"匹配度: XX分"** 绿色徽章
   - ✅ 高分候选人有 **"🤖 AI建议打招呼"** 按钮

**如果没看到徽章**：
- 刷新页面（F5）
- 打开控制台（F12）查看是否有错误
- 点击插件图标，点击"刷新分析"

#### 测试B：智能回复
1. 点击任意候选人进入聊天窗口
2. 等待对方发送消息（或查看历史消息）
3. 预期结果：
   - ✅ 输入框上方弹出 **"🤖 AI智能回复建议"** 面板
   - ✅ 显示3种风格的回复选项
4. 点击任意"使用此回复"按钮
5. 预期结果：
   - ✅ 内容自动填充到输入框
   - ✅ 出现提示："已填充回复，请检查后点击发送"

**如果没弹出面板**：
- 确保输入框为空
- 手动删除输入框内容后等待
- 刷新页面重试

#### 测试C：智能打招呼
1. 在候选人列表找到有绿色徽章的候选人
2. 点击 **"🤖 AI建议打招呼"** 按钮
3. 预期结果：
   - ✅ 弹出居中面板
   - ✅ 显示匹配度信息
   - ✅ 显示2种招呼语选项
4. 点击任意"使用"按钮
5. 预期结果：
   - ✅ 自动跳转到聊天窗口
   - ✅ 招呼语填充到输入框

## ❌ 常见问题快速解决

### Q1：插件加载失败
**错误提示**："清单文件缺失或不可读"

**解决方案**：
```bash
# 检查文件是否存在
ls boss-zhipin-assistant/manifest.json

# 检查文件权限
chmod -R 755 boss-zhipin-assistant/
```

### Q2：页面没反应
**现象**：看不到任何徽章或面板

**解决步骤**：
1. 确认插件已启用（chrome://extensions/ 查看）
2. 确认在Boss直聘页面（URL包含 zhipin.com）
3. 刷新页面（Ctrl+R）
4. 查看控制台是否有错误（F12）

### Q3：控制台报错
**错误类型A**：找不到某个文件
```
Failed to load resource: net::ERR_FILE_NOT_FOUND
```
**原因**：文件路径错误或文件缺失
**解决**：检查所有文件是否完整创建

**错误类型B**：CSS加载失败
```
Refused to apply style from... MIME type ('text/html')
```
**原因**：manifest.json 配置问题
**解决**：检查 web_accessible_resources 配置

### Q4：功能部分失效
**现象**：筛选有效，但回复建议不弹出

**原因**：Boss直聘页面结构可能有变化

**临时解决方案**：
1. 打开 F12 控制台
2. 运行以下命令手动触发：
```javascript
// 测试回复助手
autoReplyAssistant.showReplySuggestions('测试消息', document.querySelector('textarea'));

// 测试打招呼
autoGreetAssistant.scanCandidates();
```
